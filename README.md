# Customer Churn Prediction

## Overview

This project predicts whether a customer will leave a company using Machine Learning. The notebook demonstrates a complete end-to-end supervised learning workflow using Logistic Regression.

---

## Business Problem

Customer churn is one of the biggest challenges for subscription-based businesses. Retaining existing customers is cheaper than acquiring new customers. This project helps identify customers who are likely to leave.
